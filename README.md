# API - Le Bon Sandwich

##Routes

###Catégories

Accéder à une catégorie: (via un get)
>categories

Créer une catégorie: (via un post)
>categories

Modifier une catégorie: (via un put)
>categories

###Sandwichs

Accéder à la liste des sandwichs: (via un get)
>sandwichs

Paramètres possibles:
>sandwichs?type=&img=&page=&size=
